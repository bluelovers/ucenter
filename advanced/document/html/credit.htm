積分兌換請求

bool uc_credit_exchange_request(integer uid , integer from , integer to , integer toappid , integer amount)

函數參數
參數 含義
integer uid 用戶 ID
integer from 原積分
integer to 目標積分
integer toappid 目標應用ID
integer amount 積分數額
返回值
值 含義
bool 1:請求成功
0:請求失敗

接口流程

兌換參數設置
UCenter 積分兌換發起請求 API 接口 getcreditsettings 返回給 UCenter 積分項目 UCenter 中設置積分兌換參數 API 接口 updatecreditsettings 保存兌換參數
積分兌換
積分兌換界面 接口函數 uc_credit_exchange_request() 向目標應用發起兌換請求 目標應用 API 接口 updatecredit 更新用戶積分 更新本應用用戶積分
